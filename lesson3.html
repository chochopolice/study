Lesson 3

キャラクターを動かそう！

画面の中にいるキャラクターを上下左右に動かします。ゲーム作りの入り口になるレッスンです。

やってみよう

🙂
ボタンを押してキャラクターを動かしてみよう！

コードのポイント

let x = 30;
let y = 250;

function movePlayer(direction) {
  if (direction === "right") {
    x = x + 20;
  }

  player.style.left = x + "px";
  player.style.top = y + "px";
}

x は横の場所、y は縦の場所です。数字を変えると、キャラクターの場所が変わります。

ミッション

  • 1回で動く量を20から30に変えてみよう
  • 顔文字を🙂から🐱や🚗に変えてみよう
  • キーボードの矢印キーでも動かしてみよう
