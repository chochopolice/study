Lesson 4

数字を覚えよう！
へんすうってなに？

へんすうは、数字や言葉を入れておける「箱」です。
今回は、ボタンを押すたびに数字がふえるしくみを作ってみよう！

きょう やること

数字を覚える箱を作って、ボタンを押したら数が増えるようにします。

へんすう = データを入れておける名前つきの箱

できるようになること

  • 数字をしまっておく
  • 数字を 1 ずつ増やす
  • ゲームの「スコア」の基本を知る

やってみよう

いまのポイント

0

ボタンを押してみよう！

コードを見てみよう

let score = 0;

function addPoint() {
  score = score + 1;
  document.getElementById("scoreBox").textContent = score;
}
score という名前の箱に、数字を入れています。

ことばの意味

let：新しい箱を作る合図
score：箱の名前
=：右のものを左の箱に入れる

どう動いているの？

  1. score = 0 で、最初は 0 を入れる
  2. ボタンを押す
  3. score = score + 1 で数字をふやす
  4. 画面の数字も新しい数に変わる
